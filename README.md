# detlef

> Progressive Web App

```bash
# Install
$ npm i

# environment variables
$ cat > .env << EOL
CMS_DOMAIN=
APP_DOMAIN=
MAILCHIMP_API_KEY=
MAILCHIMP_LIST_ID=
MAILCHIMP_INSTANCE=
EOL

# Dev
$ npm run dev

# Build
$ npm run build

# Start
$ npm start

# Or start with process manager 
$ pm2 start process.json
```
